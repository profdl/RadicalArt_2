Interactive digital art


Environments.

User-driven exploration. Passive, static environments. The user navigates. The computer is essentially a storage device. [Typically: Network topologies: hypertext, hypermedia.]

Games.

Interactive, dynamic environments. [Often: simulated 3D environment ("virtual reality").]

Tools.

The user gets building blocks, to make whatever she wants. Typically, the space of possibilities is virtually infinite. For instance: programming languages, paint programs, virtual lego.
